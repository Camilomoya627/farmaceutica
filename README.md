#  MediStock — Sistema de Gestión Farmacéutica

MediStock es una aplicación web moderna e intuitiva diseñada para la administración eficiente del inventario y control de ventas de una farmacia. El sistema ayuda a prevenir pérdidas mediante un panel inteligente que alerta sobre productos agotados, stock bajo y medicamentos próximos a vencer en los siguientes 90 días.

---

#Características Principales

* **Panel de Control (Dashboard):** Indicadores en tiempo real con el total de inventario, productos agotados, stock bajo, ingresos del mes actual e ingresos del mes anterior.
* **Gráficas Analíticas Avanzadas:** * Gráfico de barras mensuales para el histórico de ingresos (COP).
    * Gráfico de torta interactivo que distribuye el stock por categorías.
    * Gráfico de rendimiento para identificar los medicamentos más vendidos.
* **Gestión de Inventario (CRUD Completos):** Filtrado inteligente por texto (nombre, lote o proveedor), categorías y estados de disponibilidad. Incluye formularios dinámicos para agregar y editar medicamentos.
* **Módulo de Ventas:** Registro inmediato de transacciones con validación automatizada de stock disponible y cálculo de totales.
* **Notificaciones Flotantes (Toasts):** Sistema de retroalimentación visual interactivo para confirmar acciones exitosas o alertar sobre errores.
* **Diseño 100% Responsivo:** Interfaz fluida adaptable a cualquier resolución de pantalla (móviles, tabletas y computadoras).

---

##  Tecnologías Utilizadas

* **React 18** (Librería principal de interfaz de usuario)
* **Vite** (Entorno de desarrollo rápido y empaquetador)
* **Recharts** (Generación de gráficos interactivos de barras y tortas)
* **Lucide React** (Paquete de iconografía limpia y moderna)
* **JavaScript (JSX) & CSS-in-JS** (Estilos modulares y dinámicos)
